# pnCases 1.5.0

## Что сделано

- Полностью переработан Machine GUI: компактные разделы, RGB-заголовки, динамическое обновление без мигания и безопасное закрытие интерфейсов при выключении плагина.
- Новые кейсы создаются с базовым типом `money` и собственным ключом; тип можно менять из Machine без потери привязанных блоков.
- Смена типа кейса теперь сохраняет его ключ, название, GUI, hologram, витрину, анимацию и блоки; обновляются только тип и награды.
- Загрузка и сохранение кейсов стали безопасными: дубликат ID, конфликт блоков или битый YAML не удаляют активные кейсы, а запись идёт атомарно с `.bak`-копией.
- Миграция из `config.yml:cases` в отдельные файлы проверяет полный экспорт до удаления старой секции и хранит прежние standalone-файлы в `.bak`.
- Исправлены конфликты слотов в GUI размером 9–54; Machine обновляет изменившиеся кнопки без повторной анимации открытия.
- Анимации, временно скрывающие кейс, хранят полный `BlockState` и восстанавливают блок при завершении, отмене, unload мира и выключении плагина.
- Витрина кейса упрощена до предмета, темы, стиля и переключателей; технические числовые параметры убраны из GUI.
- Выбор анимаций получил компактную сетку, короткие lore и скрытые атрибуты предметов GUI.
- Добавлена анимация «Водяная капсула»: временное стекло, сердце моря с текстурой, кристаллы призмарина, частицы воды, плавное появление награды и гарантированное восстановление блока.
- Добавлена совместимая работа GUI-обновлений через ProtocolLib при его наличии.
- Поддерживаются SQLite, MySQL и MongoDB.
- Через pnLibrary поддерживаются MiniMessage, `&`/`§`-цвета и HEX-формат `&#RRGGBB`.

## Совместимость

- Paper / Purpur 1.16.5–1.21.11
- Java 17+
- Полная «Водяная капсула» и современные display-анимации доступны на 1.21+; на старых версиях используется безопасный fallback.
