# pnCases 1.4.7

Версия 1.4.7 - небольшое обновление команд выдачи ключей. Основная цель - дать скрытно выдавать ключи, например при автоматической выдаче после покупки на сайте.

## Что нового

- У команды `/pncases givekey <игрок> <ключ> <кол-во>` появился необязательный флаг `-s` (или `-silent`).
- Если указан флаг `-s`, игрок не получает сообщение о выдаче ключа в чат.
- Ключ всё равно добавляется на баланс игрока как обычно.
- Администратору или системе, которая выполнила команду (например, сайт через RCON), по-прежнему приходит подтверждение выдачи.
- Флаг добавлен в автодополнение команды.

## Пример использования

```
/pncases givekey Evgeniy51234 donate_key 1 -s
```

Ключ будет выдан скрытно - игрок не увидит уведомление в чате, но получит ключ на баланс. Удобно для связки с сайтом: после доната сайт выполняет команду через RCON или консоль, а игрок просто видит ключ в кейсах, без лишнего спама сообщений.
