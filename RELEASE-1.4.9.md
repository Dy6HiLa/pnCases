# pnCases 1.4.9

В версии 1.4.9 стало проще настроить внешний вид наград под свой сервер. Основное внимание уделено окну «Содержимое кейса» и редкостям.

## Что нового

- Полностью настраиваемое окно «Содержимое кейса». В `gui.yml` можно изменить размер меню, расположение наград, декор, информационную кнопку, навигацию между страницами и звуки.
- Для предметов в списке наград можно настроить название, описание, показ исходного lore, свечение, Custom Model Data и порядок отображения информации.
- Добавлена настройка собственных редкостей. Для каждой редкости можно задать название, цвет, символ, описание, отдельную иконку и свечение. Разрешено добавлять свои варианты, а не только использовать стандартный список.
- Для разных типов наград можно отдельно оформить информацию: предметы, деньги Vault, поинты PlayerPoints, группы LuckPerms и команды.
- Настройка анимации теперь занимает одну кнопку. ЛКМ выбирает следующий режим открытия, ПКМ возвращает предыдущий. На Minecraft 1.16.5 автоматически остаётся доступный для этой версии «Круг фортуны».
- Внутренняя структура pnCases полностью перестроена. Работа с кейсами, наградами, интерфейсами, голограммами, хранилищем и проверкой обновлений разделена на самостоятельные части. Это упрощает дальнейшее развитие плагина и делает его поведение стабильнее.

## Совместимость

- Paper и Purpur 1.16.5–1.21.11.
- Java 17 или новее.
- LuckPerms, Vault, PlayerPoints, FancyHolograms и DecentHolograms подключаются только для соответствующих возможностей.

## Установка

1. Скачайте `pnCases-1.4.9.jar`.
2. Замените старый JAR в папке `plugins/`.
3. Полностью перезапустите сервер.
4. Пользовательские настройки и данные сохраняются. Новые параметры `gui.yml` добавляются безопасными значениями по умолчанию.
