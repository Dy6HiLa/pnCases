# pnCases 1.4.5

Версия 1.4.5 делает витрину кейса красивее и удобнее. Теперь над кейсом можно показывать свой предмет, а эффекты вокруг него включать или выключать отдельно.

## Что нового

- Добавлена живая витрина над кейсом.
- Над свободным кейсом теперь может плавно висеть и вращаться отдельный предмет.
- Предмет витрины можно поставить прямо в `/pncases machine`.
- Чтобы поменять предмет, возьмите нужный предмет на курсор и нажмите по кнопке `Предмет витрины`.
- ПКМ по `Предмет витрины` без предмета на курсоре возвращает обычный предмет кнопки кейса.
- Эффекты вокруг витрины теперь включаются отдельно.
- Можно оставить только вращающийся предмет без частиц.
- В примеры кейсов добавлены готовые настройки витрины.

## Что исправлено

- Исправлен краш частиц на новых сборках Paper/Purpur 1.21.11.
- Витрина стала спокойнее: меньше лишнего визуального шума и меньше спама частиц.
- При открытии кейса витрина скрывается, чтобы не мешать анимации открытия.
- При reload и выключении сервера временные предметы витрины корректно убираются.

## Где настраивать

Витрина настраивается в `/pncases machine` или в файле кейса:

```yaml
idle-particles:
  enabled: true
  effects: true
  style: AURORA
  theme: MAGIC
  interval_ticks: 2
  radius: 0.85
  height: 1.35
  speed: 0.14
  view_distance: 28
  item:
    material: NETHER_STAR
    name: "&aДонат кейс"
```

`enabled` отвечает за всю витрину.
`effects` отвечает только за эффекты вокруг предмета.
`item` отвечает за предмет, который будет вращаться над кейсом.
