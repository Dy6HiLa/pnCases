# pnCases 1.4.3

Версия 1.4.3 делает настройку кейсов проще и стабильнее. Главное: теперь один и тот же кейс можно поставить сразу на несколько блоков обычной командой `/pncases setcase <кейс>`.

## Что изменилось

- Теперь `/pncases setcase <кейс>` не заменяет старый блок, а добавляет новый. Можно сделать несколько одинаковых донат-кейсов, несколько кейсов с монетами и так далее.
- После перезапуска сервера привязанные кейсы больше не должны пропадать.
- Голограммы исправлены для нескольких блоков одного кейса: у каждого блока своя голограмма, и они больше не мешают друг другу.
- При открытии кейса прячется только голограмма того блока, который реально открывает игрок.
- Если у игрока нет ключей, в меню теперь нормально написано `Недостаточно ключей`, а не показывается непонятная ошибка покупки.
- Лишние команды удаления убраны. Оставлена одна понятная команда: `/pncases delcase <кейс>`.
- Добавлена новая анимация открытия: `Круг фортуны`.
- Меню настройки кейса через `/pncases machine` стало удобнее: настройки разделены по разделам, текст стал понятнее, а изменения меню обновляются корректнее.
- Содержимое кейса удобнее смотреть из основной кнопки кейса.
- Примеры кейсов вынесены в отдельные файлы `plugins/pnCases/cases/`, чтобы `config.yml` был чище и понятнее.
- Проверка старых настроек стала аккуратнее и понимает новый список блоков.
- README и manifest обновлены под новую версию.
