# pnCases 1.4.8

Скачать: [pnCases-1.4.8.jar](https://github.com/Dy6HiLa/pnCases/releases/download/v1.4.8/pnCases-1.4.8.jar)

## Что нового в 1.4.8

- Поддержка Paper и Purpur **1.16.5**.
- Поддержка MiniMessage во всех YAML-настройках: `messages.yml`, `gui.yml`, `config.yml` и файлах кейсов.
  Можно использовать старые цвета `&a`, RGB `&#42A5F5` и MiniMessage: `<red>`, `<bold>`, `<gradient:#42A5F5:#AB47BC>`.
- Анимация `Осада разбойников` для Minecraft 1.21+: четыре разбойника окружают кейс, ломают бочку, затем появляется награда.
- Отдельный `gui.yml`. В нём можно менять названия, lore, подсказки, декоративные предметы и оформление меню.
- Расширенная Machine GUI: настройка анимаций, голограммы, частицы, витрина, покупка за опыт и разметка меню кейса.
- Команда `/pncases createcase <id>` для создания нового кейса.
- Алиас `/cases` для основной команды.
- Один и тот же кейс можно поставить в нескольких местах через `/pncases setcase <кейс>`.

## Совместимость версий

- **Paper/Purpur 1.16.5:** плагин и кейсы работают, доступна одна совместимая анимация `Круг фортуны`. Современные Display-анимации и меню их выбора на этой версии не используются.
- **Paper/Purpur 1.17–1.20:** используется совместимый режим анимаций без современных Display-сущностей.
- **Paper/Purpur 1.21–1.21.11:** доступен полный режим, включая современные анимации и `Осаду разбойников`.

## Установка

1. Скачайте `pnCases-1.4.8.jar`.
2. Поместите файл в папку `plugins/`.
3. Полностью перезапустите сервер.
4. Настройте `plugins/pnCases/config.yml`, `messages.yml`, `gui.yml` и файлы в папке `cases/`.
