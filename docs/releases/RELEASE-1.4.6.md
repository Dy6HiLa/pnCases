# pnCases 1.4.6 GlobalVersion

Версия 1.4.6 GlobalVersion делает pnCases готовым для серверов Paper/Purpur 1.19 - 1.21.11. Главная цель обновления - чтобы один JAR нормально запускался на разных версиях, а админ не ловил ошибки из-за новых API Minecraft.

## Что нового

- Добавлена поддержка Paper/Purpur 1.19 - 1.21.11.
- JAR теперь собирается под Java 17, поэтому подходит для 1.19/1.20 и продолжает работать на Java 21.
- `api-version` изменён на `1.19`, чтобы старые серверы не отклоняли плагин как 1.21-only.
- При запуске pnCases проверяет версию сервера. Если версия ниже 1.19, плагин сам отключится и напишет понятную причину.
- Для 1.21+ остаются полноценные красивые анимации открытия.
- Для 1.19-1.20 добавлен совместимый режим анимаций, который не зависит от новых display/particle API.
- Витрина над кейсом стала безопаснее для разных версий и больше не привязана жёстко к `ItemDisplay`.
- Голограммы продолжают работать через FancyHolograms или DecentHolograms, а при отсутствии этих плагинов кейсы просто работают без голограммы.
- Команда `/pncases` теперь показывает единое меню: версия, статус обновления, поддержка и все команды в одном сообщении.

## Что изменилось внутри

- Общая сборка подготовлена под Java 17 bytecode.
- Настройки плагина переведены на `api-version: 1.19`.
- Для 1.19-1.20 добавлен отдельный runtime-выбор анимаций.
- Для общих элементов интерфейса и витрины добавлены безопасные варианты материалов, частиц и подсветки.
- Новые 1.21-only анимации не загружаются на старых серверах, где этих API ещё нет.

## Важно

На 1.21+ используются основные красивые анимации pnCases.

На 1.19-1.20 используется совместимый режим анимаций: кейсы открываются стабильно, награды выдаются корректно, но визуал может быть проще, чем на 1.21.
