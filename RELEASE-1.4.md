# pnCases 1.4

Версия 1.4 - большое обновление pnCases: новый preview наград, поддержка валютных наград, SQLite-хранилище, улучшенные голограммы и новая анимация открытия.

## Что нового

- Добавлена новая анимация открытия: **Астральный разлом**.
- Добавлена кнопка **Содержимое кейса** в GUI: игрок видит все возможные награды, шансы, вес шанса и редкость.
- Добавлены награды через Vault.
- Добавлены награды через PlayerPoints.
- Добавлены редкости наград: `COMMON`, `UNCOMMON`, `RARE`, `EPIC`, `LEGENDARY`, `MYTHIC`.
- Добавлено единое SQLite-хранилище `data.db` вместо отдельных `keys.yml`, `player_prefs.yml`, `open_history.yml`, `pending_rewards.yml`.
- Добавлена миграция старых YAML-файлов данных в SQLite.
- Добавлена поддержка FancyHolograms, DecentHolograms и fallback-голограмм через TextDisplay.
- Улучшена валидация старых конфигураций.
- Сделаны более тихие и плавные звуки/эффекты анимаций.

## Исправления

- Исправлена выдача VAULT / PlayerPoints / LuckPerms-наград с визуальным предметом.
- Для не-предметных наград теперь используется `visual:`, а не `item:`, поэтому золотой слиток или другой визуал больше не выдается игроку вместо валюты.
- Исправлена настройка ключа для luckperms-кейса в примерном `config.yml`.
